# agbe
Highly WIP Game Boy Advance Emulator.

## Accuracy 
 - Runs some relatively difficult-to-run games, e.g. Mario & Luigi Superstar Sage
 - Passes the [AGS Aging Cartridge](https://tcrf.net/AGS_Aging_Cartridge)
